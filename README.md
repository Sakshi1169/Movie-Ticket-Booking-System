# 🎬 Movie Ticket Booking System

> Console-Based Movie Ticket Booking Application built using Core Java, JDBC, and MySQL

---

## 📌 Project Overview

The Movie Ticket Booking System is a console-based application developed using Core Java and JDBC.  
It allows users to manage movies, shows, seats, and bookings efficiently with proper validation logic to prevent double booking.

This project demonstrates practical implementation of:
- Core Java
- OOP Principles
- JDBC Connectivity
- MySQL Database Integration
- CRUD Operations

---

## 🚀 Features

- Add, Update, Delete, and View Movies
- Manage Shows and Theaters
- Seat Booking System
- Double Booking Prevention Logic
- Booking Management
- Secure Database Queries using PreparedStatement
- Structured OOP-based Design
